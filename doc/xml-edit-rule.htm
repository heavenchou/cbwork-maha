查證各版用字所作修訂

原則是：
  • 按前後文義、所查閱版本用字、大正藏在本經的慣常用字，選擇 <lem> 修訂用字。
  • 若查閱版本用字與「修訂用字」為同字，或異體字但變形不大的，放在 <lem>，不用記錄字形。
  • 若查閱版本用字與「修訂用字」為變形較大的異體字，也是放在 <lem>，但要記錄字形。
  • 若查閱版本用字與「修訂用字」為完全不同用字，放在 <rdg>，要記錄字形。
  • 異體變形的大小，以「通用變異字形」的大原則來判斷。

假設範例：

<lb n="0087c02"/>心所欲
<app>
　　<lem wit="【CBETA】【麗】【磧】"
　　　resp="CBETA.maha"
　　　cf1="K03_p0973b12 CB01191" 
　　　cf2="Q03_p0456c11">往</lem>
     <rdg wit="【龍】" resp="CBETA.maha" cf1="L14_p0250a15">主</rdg>
     <rdg wit="【大】">住</rdg>
</app>諸佛土於佛

意思是：

  • 【大】作"住"，【麗】作"[彳*生]"(CB01191)，【磧】作"往"，【龍】作"主"，【CBETA】選擇"往"。
  • 按前後文義、所查閱版本用字、大正藏在本經的慣常用字，<lem> 採用"往"。
  • 【麗】的"[彳*生]"是"往"的異體(變形較大)，將其字形記錄在 <lem> cf 裡當做有效修訂參考。
  • 【磧】作"往"或與"往"相當的異體(變形不大，假設是"[彳*(生-(白-日))]")，也放在 <lem> 當做有效修訂參考，但不記錄其細微變化的字形。
  • 【龍】的"主"是順便查到的，記錄在 <rdg> 當做無效修訂參考。

沒做記錄的修訂

凡對底本所作修訂均做記錄, 除了:

一、標點符號的修訂

二、咒語斷句空格的修訂

例:
T08n0257_p0852c09Z##「怛[寧*也]([06]切身)他(引)(一[07]句)　唵(引)[言*我]帝(引)[言*我]帝(引[08]引)(二)　播(引)囉
=>
T08n0257_p0852c09Z##「怛[寧*也]([06]切身)他(引)(一[07]句)　唵(引)　[言*我]帝(引)　[言*我]帝(引[08]引)(二)　播(引)囉

三、大段文字位移

例: X78n1546「中卷佚文」搬家，搬至「卷上」後面。

四、悉漢對照位移

只要是悉漢對照，漢文行的行首若有「小字」就移位到上一行行尾
