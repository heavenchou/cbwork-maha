2. 簡單標記版 → XML 檔 (製作 X??.cfg，執行 Basicx.bat X??.cfg) --- Ray

2.1. 作業內容：處理程式轉換的 Basic XML 檔。

2.2. 作業項目：

2.2.1. 跑 cbent （C:\cbwork\xml\Xxx>cbent）。

2.2.2. XML語法檢查，跑 parsevol.bat（C:\cbwork\xml\Xxx>parsevol）：

處理 “xx”：包括：”?”標記、缺字找不到CB碼、偈頌是夾註等等。

其他：標記不完整。

2.2.3. 檢查XML檔中的卷首資訊--- 跑  juan.bat（C:\cbwork\xml\Xxx>juan），修訂XML檔。

2.2.4. 檢查每一卷之前是否有milestone，可搜尋 <juan fun=”open”。

2.2.5. 處理各冊 note.txt 檔的備註，修訂XML檔。

若須參照 "經前目錄" 手動插入mulu label時，請留意：

a. 若 "經前目錄" 有加上官位稱謂 或「釋」字等，則手動將它們加到 mulu label 裡。

b. 若 "經前目錄" 的用字 與 經文內的標題用字不同 (為異體字或小部份的字形有差異) ，mulu label 裡的用字以 經文內標題用字為主。

2.2.6. 修訂錯誤目錄及標記：

2.2.6.1. 搜尋“。</head>”、“。</jhead>”，將</head>、</jhead>前的圈點刪除。

2.2.6.2. 搜尋 “者</head>”，檢查是否須刪除mulu中的”者”。

2.2.6.3. 搜尋 ”音釋</head>”，刪除”音釋”的mulu。

2.2.6.4. 搜尋 “<div1><div”，檢查有無錯誤標記。

2.2.6.5. 搜尋”label=”○”、 ”label=”●”、”label=”▲”、”label=”△”， 刪除記號。

PS. 目錄中出現的全形圓圈記號”○”或”▲”、 ”△”記號，除非有特殊的區隔作用，否則將其刪除。

2.2.6.6. 檢查包含在<head>裡的校勘條目內容，是否須要修訂目錄。(標記人員手動修訂<head>內容時，記得也要修訂mulu，mulu直接修訂就好，不用 作修訂格式。 ※Maha在<head>內做的修訂，程式會在mulu自動轉成修訂用字)

ð 用正規表達式搜尋，指令：<head.*anchor.*</head>

2.2.6.7. 檢查mulu裡的夾註是否要修訂。(搜尋 “) "/><head>”)

2.2.7. 處理卷與卷交界的重複標題：

Step1. 將 .xml檔 copy到該冊xml各經統計裡，運用 Spcl2_4.exe（特殊字串頻次統計程式），設定頭 ("<mulu") 尾 ("/>") 字串，用多檔頻次統計出各經頻次大於"2"的重複標題。

Step2. 檢查所列的重複標題是否須修訂。

Step3. 修訂方式：(將 <div> <mulu> <head> 改成 <p id=”” type=”head\d+”>)

Ø 在重複標題的<lb>前加上#，並刪除上一層div，以符合巢狀結構。(取代前先不作儲存)

Ø 做多重檔案取代 (檔案/類型：*.xml)，取代動作：

取代指令：(正規表達式請打勾)
#(<lb ed=".*") n="(.*)"/><div(\d+).*><mulu.*"/><head>(.*)</head>
取代成：(請改冊數參數)
\1 n="\2"/><p id="pX79p\201" type="head\3">\4</p>

3. 檢查 HTML Help 目錄

3.1. 作業內容：檢查產生的目錄、目錄層次，修訂XML檔。

3.2. 作業項目：HHC檔製作。

3.2.1. XML→HHC 檔--- 跑hh3.bat。（C:\cbwork\xml\Xxx>hh3 Xxx）
輸出位置：C:\Release\htmlhelp

3.2.2. 打開Xxx.hhp檔，compile 成Xxx.chm檔。

3.2.3. 檢查Xxx.chm檔，修訂XML檔。（也可直接檢查XxxToc.hhc檔）

檢查目錄結構，包含：經號、經名、目錄及卷內容。

檢查本文作目錄標題是否正確，文字若不可分割，請修訂mulu標題，並將<head>改成段落首行縮排 <p id="pXxxpxxxxbxxxx" rend="margin-left:0;text-indent:x"> 比照Q空格處理(此作法是為了與sm比對時不形成空格差異)。不可分割文字如：「爾時○○尊者」、「維摩會上」。

ð 請注意段落的圈點問題：去掉head及行中段落時XML要補上圈點，段末也要加上圈點。

用字不同的重複標題，若內容相同時，刪除<div><mulu><head> 改成 <p id=”” type=”head\d+”>，若內容不同時，則保留不刪。

用字不同的重複標題，依子目錄的內容不同，可在mulu加入文字以做區別。例如：在目錄標題後加入（旁出）、（南嶽）、（青原），手動加入的目錄內容以全形小括號表示。　　

目錄中有過長的空白時，視情況將空白刪除，或只留一個半形空白。例如：「天聖廣燈錄都帙目錄　　　〔宋實〕」，修訂成「天聖廣燈錄都帙目錄 〔宋實〕」。

若為明顯的兩個標題，則中間空一個全形。

例如：X87n1623_p0227b16Q#1(虞)摩騰入漢　僧會來吳

PS. 這部份只處理「書前目錄」或「經前目錄」所提到的層次，不用做太細。
