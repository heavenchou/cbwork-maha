CBETA XML P5 標籤集

 最近更新日期： 2009-05-07

CBETA XML 文件目前使用 utf-8 Encoding， 標記採用國際標準 TEI， 在 TEI 塈鉹ㄗ鴞X適的才會自訂標記 ， 或自定標記屬性 。

一個 CBETA XML 檔內部架構分為 檔頭（Header） 及 本文（Text） 兩部份：

 

一、   XML 經文檔的整體架構介紹

 

Element 元素實例

<TEI>
　　<teiHeader>
　　　　<text>
　　　　　　<body> 正文
　　　　　　<back> 正文後資訊

<?xml version="1.0" encoding="utf-8"?>
<TEI xmlns="http://www.tei-c.org/ns/1.0" xml:id="X11n0267">
<teiHeader>......</teiHeader>
　<text>
　　<body>......</body>
　　<back>......</back>
　</text>
</TEI>

 

 

二、依檔頭、本文兩大類介紹 CBETA 目前所使用的標記
1. 檔頭（Header）

檔頭標籤集分成以下三類：

  • File Description 本電子檔書目資訊描述

  • Encoding Description 電子版與來源之間的關係描述

  • Profile Description 文件背景描述

  • Revision Description 修訂摘要記錄
