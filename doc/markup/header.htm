CBETA XML P5 標籤　檔頭（Header）

by 淑儀  更新日期：2009/06/23

 

檔頭標籤集分成以下四類：

  • File Description 本電子檔書目資訊描述
  • Encoding Description 電子版與來源之間的關係描述
  • Profile Description 文件背景描述
  • Revision Description 修訂摘要記 錄

 

Element 元素
實例
<teiHeader>
底下均以 T18n0850 為實例

<profileDesc> 文件背景描述
　<langUsage> 語言使用
　　<language> 個別敘述文本中使用的語言

<profileDesc>
　 <langUsage>
　　<language ident="sa-Sidd">Sanskrit-Siddam</language>
　　<language ident="en">English</language>
　　<language ident="zh">Chinese</language>
　</langUsage>
</profileDesc>

　　<change when> 變更內容與日期
