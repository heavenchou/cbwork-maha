CBETA XML 標籤集

  最近更新日期： 2008-11-12

CBETA XML 文件目前使用 Big5 Encoding， 標記採用國際標準 TEI， 在 TEI 塈鉹ㄗ鴞X適的才會自訂標記 ， 或自定標記屬性 。 一個 CBETA XML 檔內部架構分為 檔頭（Header） 及 本文（Text） 兩部份：

　

一、   XML 經文檔的整體架構介紹

 

Element 元素

範例

< TEI.2 >

< ?xml version= " 1.0 " encoding= " big5 " ? >

< ?xml:stylesheet type= " text/xsl " href= " ../dtd/cbeta.xsl " ? >

< !DOCTYPE TEI.2 SYSTEM " ../dtd/cbetaxml.dtd” >

< TEI.2 >

　 < teiHeader > … < /teiHeader >

　 < text >

　　 < body > … < /body >

　 < /text >

< /TEI.2 >

 

< teiHeader >

 

< text >

 

 

< body >

 

 

二、依檔頭、本文兩大類介紹 CBETA 目前所使用的標記

1. 檔頭（Header）

檔頭標籤集分成以下三類：

  • File Description 本電子檔書目資訊描述

  • Encoding Description 電子版與來源之間的關係描述

  • Revision Description 修訂摘要記錄

Element 元素

範例

2. 本文（Text） 裡   < body >   常用標籤與屬性：

分成以下七類：

  1. 最上層結構標籤集

  2. Paragraph Level 段落層標籤集

  3. Phrase Level 片語層

  4. Milestone tags 里程碑

  5. 校勘

  6. Graphics 圖形

  7. 表格

　

　

1. 最上層結構標籤集

　

　

Element 元素

範例

< div1 > … < div7 >

< text > < body >
< div1 type= " fen " n= " 1 " > 瑜伽師地論卷第一
　 　 < byline type= " Author " > 彌勒菩薩說 < /byline >
　 　 < head > 本地分 < /head >
　 < div2 type= " di " n=”1” > < head > 五識身相應地 < /head >
< div3 type= " zhang " n= " 1 " > < head > 十七地總說 < /head >
　　 …
< /div3 >
　　 …
　 < /div2 >
　 …
< /div1 >
< /body > < /text >
