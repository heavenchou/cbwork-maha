cb 欄位

# 開頭──取消使用
a 開頭──幾乎多是 CBETA 還沒有處理校勘時的校勘用字
b 開頭──國科會專案用字 [心如工作用字，應該是T31哪一個專案] (如果 CBETA 還沒有組字的話)
c 開頭──法鼓全集用字(如果 CBETA 還沒有組字的話)
n 開頭──國科會專案用字和西蓮的南山律用字 [心如、葛賢敏] (如果 CBETA 還沒有組字的話)
x 開頭──CBETA 通用詞用字。原本四碼，用五碼後決定 x+通用詞的缺字cb碼(如果詞組都是缺字，採用首字的cb碼)，前後沒有統一，所以有一個x和兩個x。

通用詞

通用詞 於 XML 中使用的 entity, 例如 CI0001, 記錄於 entity 欄位.
