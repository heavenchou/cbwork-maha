<p=h1>

<p=h1> 表示一個段落性的重複標題，標題層次為1, XML 轉成 <p type="head1">

Q

標題之間允許空白行

# X01n0026_p0417c07Q#1大梵天王問佛決疑經
# X01n0026_p0417c08Q=1
# X01n0026_p0417c09Q=1凡例

標題含夾註

標題本身是夾註，轉出的 XML mulu label 裡的半形括號要拿掉。

SM: X59n1084_p0336b13Q#1(入禪定偈)

XML: <mulu type="其他" level="1" label="入禪定偈"/><head><note place="inline">入禪定偈</note></head>

但是夾註本身是標題的一部份，這種的半形括號就要保留。

<mulu type="其他" level="2" label="(附)略法界觀門頌"/><head><note place="inline">附

※ sm 要用到 m= 的，統一用 <Qn....> 不用 Q#1，而原來標記欄位的標記改標在 <Qn....> 後面。

W

1. 遇到 W, 會往下開一層 div. 如果希望它開 div2, 那就加 </Q2> 把目前 div2 結束, 回到 div1, 遇到 W 就會開 div2.

2. W 開始了新 div, 遇到沒有 W 了, div 就會結束, 不用特別加 div 結束標記, 如 </Q2>.
