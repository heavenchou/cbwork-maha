<entry> 辭書 或 註釋書(原文段落名相註解、字詞解釋)

2008/01/22

Description

  • 用於辭書(段落後註解、隨文註解) 或 註釋書(原文段落名相註解、字詞解釋)。

  • 辭書 <entry> 是一整個辭條，辭條裡面包含 辭 <form> 及 解釋 <def>。

  • 「註釋」有以下類型：

    1.     列出原文段落中名相，再逐條解釋。(BM: n d p；XML: <entry><form>...<def>)
    2. 原文 逐句 or 逐段 解釋。(BM: o u p；XML: <div1 type="orig"><p>...<div1 type="commentary"><p>)

    註釋書 <entry> 的型式是「原文段落名相註解」、「字詞解釋」，因為此種段落後註解，是夾在段落之間，所以不用 div1，改用下層 div。


Attributes

place

  • inline 行中辭書、註釋書：辭書、註釋書起始點不在行首

rend

  • 辭書、註釋書段落縮排

　

May Contain (可以包含的子元素)

<form>  <def>
