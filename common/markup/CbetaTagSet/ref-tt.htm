<tt> 詞組對照

2008/03/03

Attributes

n

如果這個勘誤資訊是大正藏校勘欄提供的, n 屬性的前四碼是校勘出處的頁碼, 後三碼是校勘編號.

rend

  • inline: 行中對照, 例：T36n1736_p0435b23, T54n2135_p1223a27
  • normal: 按原大正藏分行對照. 例：T54n2133A

type

  • app: 由校勘資訊轉成的 <tt>
  • inline: 改用 rend="inline"
  • tr: translation 翻譯對照.例：T20n1168B

word-count:

type="app" 時的字數
　

May contain (可以包含的子元素)

<t>

Examples

  • 行中對照

  • 一行漢字、一行悉曇字

  • 雙行單詞對照

  • 三行整行對照

  • 校勘的 <tt> (type="app")

  • <tt> 下含 <app>

  • word-count
