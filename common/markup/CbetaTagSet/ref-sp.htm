<sp> 發言、述說

2008/01/22

Description

有規則的問答段落 或 陳述、辯證裡面的一組發言、述說。
　

Attributes

type

  • type="question" (有規則的問答段落 型式－－發問)
  • type="answer" (有規則的問答段落 型式－－回答)
  • type="statement" (陳述、辯 證型式－－陳述)
  • type="argument" ( 陳述、辯 證型式－－辯 證)


May occur within (可以被包含在元素之內)

<dialog>
