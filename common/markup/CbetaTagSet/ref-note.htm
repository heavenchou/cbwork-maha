<note> 註

$Date: 2005/03/24 08:41:51 $

有些校勘條目，以注解的方式注明漢梵版本資訊、出處或收藏地（者），而該筆校勘條目，不能以<app>標記定型式來標記，就必須以<note type="foot" n="?" >…</note>(或<note type="foot">…</note>)、<note type=" inline " n="?">…</note>或<note type=" sk " n="?">…</note>標記該筆校勘條目。
<note type="foot" n="?" >…</note>有時標記經文的「夾註」字。

  • Attributes

  • 範例
    • resp="CBETA"
    • type="inline"
    • type="foot"
    • type="sk"
    • <note> 在 <app> 的 <lem>
    • <note> 在 <app> 的 <rdg>
  • 特例
    • 一筆校勘兩個 <note type="orig">

Attributes

lang

n

  • <note type="orig"> 的 n 屬性通常不會重複, 除非如 T02, No. 99, p. 170, c17, 一條校勘在內文有兩處校勘符號, n 屬性的結尾會加 A, B.

place 註解所在位置 或 校勘數字符號 所在位置

  • inline: 行中雙行夾註

  • foot: 出現在頁尾的註, 也就是大正藏的校勘欄

  • interlinear: 側註, 傍註, 行與行之間的註, 出現在續藏

如果 type="orig", 則 place 屬性表示 校勘數字符號 出現在哪

  • place="foot text" 表示 頁尾 跟 內文 堻ㄕ陵桹冱あr符號

  • place="foot" 表示 頁尾 有 校勘數字符號, 但是內文堥S有對應的.

  • place="text" 表示 內文 有 校勘數字符號, 但是頁尾堥S有對應的.

resp (i.e. responsible) 負責人

  • <note resp="Taisho" place="foot">....</note> 記錄大正藏的校勘原文
  • resp="CBETA"
    • 記錄 CBETA按
    • CBETA 修訂過的 note, 例如:
      T13p0391
      <lb n="0391a21"/>來所出家
      <note n="0391007" resp="Taisho" type="orig" place="foot text">剔＝剃【宋】【明】【宮】，　【元】，【明異】</note>
      <note n="0391007" resp="CBETA" type="mod">剔＝剃【宋】【明】【宮】，剔＝&lac-space;【元】，【明異】</note>
      <app n="0391007"><lem>剔</lem><rdg wit="【宋】【明】【宮】" resp="Taisho">剃</rdg></app>
      <note n="0391007" resp="CBETA" type="rest">剔＝<todo/>&lac-space;【元】，【明異】</note>髮一切諸佛之所住持興三寶

type 註解的類別

  • inline: 舊的用法, 改用 place="inline"

  • foot: 舊的用法, 改用 place="foot"

  • mod: modified, 經 CBETA 修訂的大正藏校勘

  • orig: original, 大正藏校勘原文

  • rest: 由原大正校勘取出的註解

  • sk

範例 (type="inline")

如《大明度經》卷第一(T08.225.479a)的「性與道俱如合符契也」等字，為經文中的「夾註」。以<note type="inline">性與道俱如合符契也</note>標記。

<lb n="0479a21"/>不驚不怛不以恐受。以體解而性入
<note type="inline">性與道俱如合符契
<lb n="0479a22"/>也</note>是為住不退轉。應於無處當以知此。又妙

範例 (type="foot")

如果是校勘條目的注解文，如在T30.1579.478b的第二（[02]）校勘碼注解著「梵本 Bodhisattva-bh&umacron;mi 京都帝國大學所藏」，此時以<note type="foot" n="047802">梵本 Bodhisattva-bh&umacron;mi 京都帝國大學所藏</note>標記該校勘條目的內容，置放在卷首《瑜伽師地論》卷第三十五之前。

範例 (note在app的lem)

假設該筆校勘條目在大正藏是「夾註」，而在校勘條目則注明「作本文」，如《大般涅槃經》卷第十八（T12.375.723c）「是故[18]名為為阿闍世」的「名為」二字，以在宋元明三本是「作本文」則以<lem><note type="inline">名為</note></lem>標記該二字，<rdg >寫成<rdg wit="【三】">名為</rdg>即可。
