範例：【下同與＊】

大正藏經文中經常可以在經文中或校勘條目中看到「＊」的校勘符號或「下同」二字的注明。在大正藏略符中，「＊」的校勘符號表示「下同」（so below, so above）的意思。經文或校勘條目中有時用「＊」，有時用「下同」二字。

在XML電子檔中，以<app>標記第一筆的校勘字，如果第二筆的校勘字出現「＊」，就得重覆第一筆的校勘條目之資訊。如《般若波羅蜜多心經》「揭[04]帝揭[05] 帝」的「帝」字，宋元明三本作「諦」字，該「帝」字在下一句「般羅揭＊帝」出現，該「帝」字卻以「＊」表示，此時以校勘代碼「id="p0848c01"」表示這「＊」在這一頁（T08.251.848c）是第一個「＊」的校勘碼，並重複校勘碼[05]的校勘資訊。

<app> 修訂


Description

一般使用大正藏的使用者，所用有新文豐、世樺、白馬等出版社的所印的大正藏影印本。而這些大正藏影印本，在處理經文的用字上，常常有貼錯等現象。另一情形是日本原版大正藏已經是錯別字了。

CBETA 在校對這一方面，是收集三處他人所輸入的電子檔進行比對，在比對當中所出現的差異字，經過校勘相關版本後，以<app><lem>...</lem><rdg>...</rdg></app> 標記訂正該差異字。在 <app> 中 <lem> 是 CBETA 修訂用字，<rdg> 是原文字，有 "resp" 屬性則記錄該訂正者的名字。

Attributes

  • resp(負責人), cf1(參考資訊), cf2(參考資訊), wit(版本)

  • 若是兩個負責人，中間空一個半形 resp= "CBETA.xxx CBETA.xxx" 來表示。

範例：

如《大方廣佛華嚴經》卷第三（T09.278.411a）的「依止處空住」的「處」字，經過 "maha" 君回查高麗藏及蹟砂藏時，校勘出「處」字應為「虛」字才對，因此以

<app><lem resp="CBETA.maha" cf1="K08n0079_p0019c05 虛" wit="【CBETA】【麗】【磧砂】" cf2="Q07_p0361c20 虛">虛</lem><rdg wit="【大】">處</rdg></app>
